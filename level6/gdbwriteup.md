```as
(gdb) x/s 0x80485b0
0x80485b0:       "/bin/cat /home/user/level7/.pass"
```
This function calls the system function with the string "/bin/cat /home/user/level7/.pass" as argument. This means that if we can overwrite strcpy function in the `main` with the address of the function `n` we will be able to read the password of the next level.
